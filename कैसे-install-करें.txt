STEP 1 — इस फ़ोल्डर (BahikhataApp) को ZIP करें
──────────────────────────────────────────────
Windows Explorer में BahikhataApp फ़ोल्डर पर
Right-click → "Send to" → "Compressed (zipped) folder"

STEP 2 — GitHub Pages पर होस्ट करें (सबसे आसान, मुफ़्त, टिकाऊ)
────────────────────────────────────────────────────────────
 A. github.com पर जाएँ → मुफ़्त account बनाएँ (अगर नहीं है)

 B. ऊपर "+" → "New repository" दबाएँ
    - Repository name: bahikhata
    - Public चुनें
    - "Create repository" दबाएँ

 C. "uploading an existing file" link दबाएँ
    BahikhataApp फ़ोल्डर के अंदर की सभी फ़ाइलें
    और फ़ोल्डर (icons/, server/) चुनकर Upload करें

 D. "Commit changes" दबाएँ

 E. Settings → Pages → Source: "main" branch, root (/)
    → Save

 F. कुछ मिनट में लिंक मिलेगा:
    https://आपका-नाम.github.io/bahikhata/

STEP 3 — फ़ोन में ऐप install करें
───────────────────────────────────
 A. ऊपर वाला लिंक अपने Android फ़ोन के
    Chrome browser में खोलें

 B. Chrome में ऊपर ⋮ (तीन बिंदु) दबाएँ
    → "Add to Home screen" या "Install app"
    → "Add" दबाएँ

 C. हो गया! होम स्क्रीन पर बहीखाता आइकन
    आ जाएगा — बिल्कुल app की तरह।

दूसरा तरीका — असली APK फ़ाइल (Play Store स्टाइल):
───────────────────────────────────────────────────
 A. GitHub Pages लिंक मिलने के बाद
    pwabuilder.com खोलें

 B. वो लिंक डालें → Start → Android चुनें
    → Generate Package

 C. APK फ़ाइल डाउनलोड होगी

 D. अपने Android फ़ोन में भेजें (WhatsApp/email)
    → Settings → "Install unknown apps" की
      अनुमति दें → Install करें

किसी और को लिंक भेजना:
─────────────────────
GitHub Pages वाला लिंक WhatsApp/SMS में
भेज दें — कोई भी उसे अपने फ़ोन में खोलकर
"Add to Home screen" से install कर सकता है।
हर किसी का अपना अलग डेटा होगा।

STEP 1 — मुफ़्त Gemini key लें
──────────────────────────────
 A. aistudio.google.com/apikey खोलें
 B. Google account से sign in करें
 C. "Create API key" → "AIza..." key कॉपी करें
    (बिल्कुल मुफ़्त, कोई credit card नहीं)

STEP 2 — ऐप में key डालें
───────────────────────────
 A. ऐप खोलें → ऊपर ⚙️ बटन दबाएँ
 B. "अपनी मुफ़्त Gemini key" में key पेस्ट करें
 C. सेव करें

STEP 3 — फोटो से entries भरें
──────────────────────────────
 A. ऐप में "📸 फोटो/बिल से entries भरें" दबाएँ
 B. रजिस्टर का एक पन्ना फोटो खींचें
 C. "🔍 पढ़ना शुरू करें" → review करें → "जोड़ें"

नई entry जोड़ना:
  ＋ बटन → नाम, रकम, दिया/लिया चुनें → सेव

किसी का हिसाब WhatsApp पर:
  "लोग" टैब → उसके आगे 🟢 WhatsApp बटन

तारीख़-वार हिसाब:
  "📅 तारीख़ वार" टैब → किसी भी दिन का 🟢 बटन

नाम सुधारना (एक बार में सब entries):
  "लोग" टैब → नाम के आगे ✏️ बटन

फोटो से entries पढ़ना:
  "📸 फोटो/बिल से entries भरें" बटन

बैकअप (ज़रूरी — हर हफ़्ते):
  नीचे "💾 बैकअप फ़ाइल सेव करें" → Google Drive
  या खुद को WhatsApp पर भेजें

डेटा वापस लाना (नए फ़ोन पर):
  ऐप install करें → "♻️ बैकअप से वापस लाएँ"
  → वही .json फ़ाइल चुनें
